NOTEWORTHY:

The technologies that I used include Bulma CSS and Vanila JS. By following some tutorials online, I was
able to create a project which I both feel is my own and worth the time to look at!

There are multiple visualizers available for viewing and some different checkboxes to change the composition.
By choosing one of the mp3 files in the select box, the user can enjoy a listening experience like no other!

GRADING:

I met project requirements by using BULMA CSS. I cited all of the sources that I used
for this project as well as making a large of canvas for an enjoyable view.
In my opinion, the homework regarding the visualizer was a good start.. but it
did not quite "wow" me. After looking online, it seems as though different libraries
are a good way to introduce oneself to more audio visualization.

I went above and beyond by adding different a couple different choices in filters
which required geometric manipulation through shapes and color.
However, I was not able to finish the file addEventListener nor the JSON.
It would have been nice to link these together for a more user-interactive experience.
I had quite a few ideas that I was not able to add in time for this project.
I think that with more research and practice I can make something using libraries
as well as Adobe After Effects in the future.

I want to get involved with more audio visualization for music. This is a really cool type of coding!
