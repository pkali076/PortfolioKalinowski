Rebane's (Modified by pkali076) Discord Colored Text Generator

This app formats colored text Discord messages using the ANSI color codes available on the latest Discord desktop versions.

  • 1. Type Text (can copy and paste text into the textarea)
  • 2. Select Text (can select parts of text to alter colorings at one's will)
  • 3. Select a color above.

  • Make sure to select a color afterwards, otherwise the encoding doesn't format the text to the desired colors

  • 4. When finished with altering color, click on 'Copy Formatted Text' button
  • 5. Paste text in Discord. The color will be encoded with the text; expect to see some extra symbols
